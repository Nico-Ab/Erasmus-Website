# UI Style

## Overall tone
The portal should look like a serious internal university administration system.

## Design direction
- calm
- clean
- structured
- professional
- information-dense but readable
- not playful
- not trendy startup-like
- not flashy

## SWU inspiration
Use the SWU site as inspiration for institutional tone and structure:
- clear hierarchy
- formal presentation
- navigation-first information layout
- official-looking content blocks
- restrained visual language

Do not copy the SWU site directly.
Do not mirror exact layouts or branding details.
Use it only as stylistic guidance.

## Layout
- desktop-first admin layout
- sidebar or structured navigation for internal sections
- clear page titles and breadcrumbs
- filter bars above tables
- readable table density
- obvious action buttons
- clear empty states
- clear success and validation states

## Components
- professional login page
- formal dashboard cards
- readable data tables
- compact but usable forms
- status badges with restrained colors
- review/comment panels
- document lists with version markers

## Typography
- clear hierarchy
- avoid oversized marketing typography
- prioritize readability over style

## Visual restraint
Avoid:
- large hero sections in the authenticated area
- illustrations
- startup gradients
- excessive animations
- playful cards
- gimmicky microcopy

## UX priorities
- speed of understanding
- low ambiguity
- easy review workflows
- easy scanning in tables
- clear status communication
